# 2D Lid-Driven Cavity Flow Solver using a Staggered Grid and SOR Method

This project implements a two-dimensional incompressible lid-driven cavity flow solver in Python. The solver is based on a staggered-grid finite difference method, a projection method for pressure-velocity coupling, TDMA-based implicit treatment of viscous terms, and an SOR iteration for the pressure Poisson equation.

The main purpose of this repository is to demonstrate the implementation of a CFD solver from scratch, including discretization, boundary conditions, pressure correction, and post-processing.
